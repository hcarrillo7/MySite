Carrillo Concrete, Inc

The Beginning...

CarrilloConcreteInc.com was my first web project and hits close to home, literally, as it's my Dad's Concrete Company. Initially, it started as a WordPress template using various extensions as I attempted to learn the ins and outs of domain names, hosting and all the different options available. Soon after, I felt I had the knowledge and confidence to complete a full website from beginning to end and this was the result. My goal was to create a site with a simple yet straightforward and modern design.

Tools Used: HTML, CSS, Javascript, Bootstrap
