Adventures of Val and Tia

The Gamer Inside...

Adventures of Val and Tia was a game created for my CS 485 Game Programming class. I had the pleasure to work with my two partners, Jessica Jones and Vincent Phongsamran. Coming into this class we had no experience or even a clue on how to make a video game but as the semester progressed the game began to come to life. All art was original for this game, including Textures, Characters and Animations along with all the code to make the game work as intended. The programming was done using C# and the art was created using Blender, everything finally came together using the Unity game engine.

Tools Used: C#, Unity, MonoDevelop, Visual Studio, Blender
